# Template chung — Văn bản hành chính

Áp dụng cho các loại VB chưa có file reference riêng: chỉ thị, hướng dẫn, phương án, đề án,
giấy giới thiệu, giấy nghỉ phép, phiếu gửi, thư công, v.v.

## Cấu trúc chuẩn

```
[HEADER 2 CỘT]
Cột trái:                              Cột phải:
TÊN CQ CHỦ QUẢN (nếu có)              CỘNG HÒA XÃ HỘI CHỦ NGHĨA VIỆT NAM
TÊN CQ BAN HÀNH                           Độc lập – Tự do – Hạnh phúc
     ────────                           ─────────────────────────
Số: .../[Viết tắt loại]-[CQ]           Địa danh, ngày ... tháng ... năm ...

                    TÊN LOẠI VĂN BẢN
              [Trích yếu nội dung]

[PHẦN CĂN CỨ — nếu cần]
Căn cứ ...;     ← in nghiêng, chấm phẩy
Căn cứ ....     ← dòng cuối: dấu chấm

[NỘI DUNG CHÍNH]
Trình bày theo đoạn văn hoặc chia mục I, II, III
hoặc chia Điều 1, Điều 2... (tùy loại VB)

Kết thúc bằng "./."

[FOOTER 2 CỘT]
Nơi nhận:                              CHỨC VỤ NGƯỜI KÝ
- ...;                                 (Ký, đóng dấu)
- Lưu: VT.                             Họ và tên
```
