# Cấu trúc Trình chiếu (PPTX Structure)

Quy chuẩn bố cục, phân cấp nội dung, layout cho file PowerPoint.

---

## Kiến trúc slide deck chuẩn

| Vị trí | Loại slide | Nội dung |
|---|---|---|
| Slide 1 | Title | Tiêu đề + phụ đề + logo |
| Slide 2 | Agenda / Outline | Mục lục nội dung |
| Slides 3-N | Content | Nội dung chính |
| Slide N+1 | Summary / Key Takeaways | Tóm tắt 3-5 điểm chính |
| Slide cuối | Thank you / CTA | Thông tin liên hệ |

---

## Nguyên tắc bố cục slide

### Quy tắc 1 — Không slide thuần text
Mỗi slide phải có **ít nhất 1 yếu tố thị giác**: ảnh, biểu đồ, icon, shape, hoặc số lớn.

### Quy tắc 2 — Cỡ chữ tối thiểu
| Vai trò | Size tối thiểu |
|---|---|
| Title slide | 36-44pt |
| Section header | 24-32pt |
| Content heading | 20-24pt |
| Body text | 14-16pt |
| Caption / source | 10-12pt |

### Quy tắc 3 — Margin & spacing
- Margin tối thiểu: 0.5 inch (1.27 cm) các cạnh
- Gap giữa content blocks: 0.3-0.5 inch

### Quy tắc 4 — Không lặp layout
Vary giữa các slide: two-column, icon grid, stat callout, half-bleed image...

---

## Layout patterns

| Pattern | Mô tả | Dùng cho |
|---|---|---|
| **Two-column** | Text trái + illustration phải | Giải thích concept |
| **Icon rows** | 3-4 icon trong circle + label | Liệt kê tính năng |
| **2×2 Grid** | 4 box ngang nhau | So sánh, phân loại |
| **Half-bleed image** | Ảnh chiếm 50% slide + text overlay | Storytelling, case study |
| **Stat callout** | Số lớn 60-72pt + label nhỏ dưới | KPI, metric nhấn mạnh |
| **Timeline / flow** | Mũi tên nối các bước | Quy trình, roadmap |
| **Quote slide** | Text lớn italic + attribution | Trích dẫn ấn tượng |

## Font pairing

Không default Arial. Chọn cặp font có cá tính:

| Heading | Body | Tông |
|---|---|---|
| Georgia | Calibri | Classic formal |
| Arial Black | Arial | Bold modern |
| Cambria | Calibri | Warm professional |
| Impact | Arial | High energy |
| Montserrat | Open Sans | Tech startup |

---

## Phân cấp thông tin trên slide

| Cấp | Style | Ví dụ |
|---|---|---|
| Title | Bold, màu Primary, 24-32pt | "Kết quả Phase 1" |
| Subtitle | Regular, màu Secondary, 16-20pt | "Tháng 5/2026 — Viettel Construction" |
| Bullet | Regular, 14-16pt | "- Hoàn thành 95% mục tiêu" |
| Nhấn mạnh | Bold + màu Accent | "95%" |
| Source / note | Italic 10pt, góc dưới phải | "Nguồn: Báo cáo nội bộ Q1/2026" |

---

## Điều cấm

| Điều cấm | Lý do |
|---|---|
| Không copy-paste bảng Word vào slide | Bể format, không editable |
| Không dùng accent line dưới title | Dấu hiệu slide AI generic |
| Không để default placeholder text | "Click to add text" = amateur |
| Không quá 6 bullet trên 1 slide | Quá tải thông tin |
| Không dùng animation phức tạp | Gây phân tâm, chậm trình chiếu |
