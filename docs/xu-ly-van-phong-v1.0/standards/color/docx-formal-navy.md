# Formal Navy — Bộ màu trang trọng

Dùng cho đề xuất cấp chiến lược, tài liệu gửi ban tổng giám đốc, tập đoàn, khách hàng lớn.

---

## 1. Bảng màu chính

| Vai trò | Mã HEX | Mô tả |
|---|---|---|
| Primary | `#1F3864` | Navy — dùng cho heading cấp 2, header bảng |
| Accent | `#C00000` | Đỏ đậm — heading cấp 1, đường viền nhấn |
| Secondary | `#595959` | Xám — subtitle, caption, metadata |
| Body text | `#202020` | Đen nhẹ (không dùng #000000 vì chói) |
| Border / line | `#BFBFBF` | Xám nhạt — header/footer line |

---

## 2. Áp dụng cho Heading

| Cấp | Color | Style |
|---|---|---|
| Heading 1 | `#C00000` (Accent) | Bold + caps + border dưới |
| Heading 2 | `#1F3864` (Primary) | Bold |
| Heading 3+ | `#202020` (Body) | Bold |
| Caption | `#595959` (Secondary) | Italic |

---

## 3. Áp dụng cho Bảng

| Vai trò | Mã HEX |
|---|---|
| Header fill | `#1F3864` (Primary) |
| Header text | `#FFFFFF` |
| Zebra row lẻ | `#F2F2F2` |
| Zebra row chẵn | `#FFFFFF` |
| Dòng tổng | Bold, không fill |

---

## 4. Áp dụng cho Callout / Block đặc biệt

| Loại | Màu chữ | Ghi chú |
|---|---|---|
| Nguyên tắc | `#1F3864` (Primary) | Bold italic |
| Điểm mấu chốt | `#C00000` (Accent) | Bold italic |
| Lưu ý | `#595959` (Secondary) | Italic |

---

## 5. Màu phân loại (Traffic Light)

| Cấp | Fill | Ý nghĩa |
|---|---|---|
| Safe / Cấp 1 | `#E2F0D9` (lá nhạt) | An toàn, cho phép |
| Warn / Cấp 2 | `#FFF2CC` (vàng nhạt) | Cần thận trọng |
| Danger / Cấp 3 | `#FADBD8` (đỏ nhạt) | Cấm, nhạy cảm |
