# Editorial Burgundy — Bộ màu editorial

Dùng cho tài liệu review, nghị luận, phản biện, đánh giá chuyên sâu.

---

## 1. Bảng màu chính

| Vai trò | Mã HEX | Mô tả |
|---|---|---|
| Primary | `#7B241C` | Burgundy — title, heading cấp 1 |
| Accent | `#8B4513` | Nâu — đường trang trí, nhấn |
| Secondary | `#555555` | Xám — subtitle, ghi chú |
| Body text | `#1A1A1A` | Đen hơi xám, dịu mắt |
| Border / line | `#C0C0C0` | Xám nhạt |

---

## 2. Áp dụng cho Heading

| Cấp | Color | Style |
|---|---|---|
| Heading 1 | `#7B241C` (Primary) | Bold |
| Heading 2 | `#8B4513` (Accent) | Bold |
| Heading 3+ | `#1A1A1A` (Body) | Bold |
| Caption | `#555555` (Secondary) | Italic |

---

## 3. Áp dụng cho Bảng

| Vai trò | Mã HEX |
|---|---|
| Header fill | `#7B241C` (Primary) |
| Header text | `#FFFFFF` |
| Zebra row lẻ | `#F5F0F0` |
| Zebra row chẵn | `#FFFFFF` |
| Dòng tổng | Bold, không fill |

---

## 4. Áp dụng cho Callout / Block đặc biệt

| Loại | Màu chữ | Ghi chú |
|---|---|---|
| Nguyên tắc | `#1B5E4B` (lá đậm) | Bold italic — khẳng định |
| Điểm mấu chốt | `#8B4513` (Accent) | Bold italic |
| Lưu ý | `#555555` (Secondary) | Italic |
| Cảnh báo | `#9B2D1F` (cam đỏ) | Bold italic — cảnh báo |

---

## 5. Màu phân loại (Traffic Light)

| Cấp | Fill | Ý nghĩa |
|---|---|---|
| Safe / Cấp 1 | `#E8F5E9` (lá nhạt) | Khẳng định |
| Warn / Cấp 2 | `#FFF3E0` (cam nhạt) | Cần xem xét |
| Danger / Cấp 3 | `#FFEBEE` (đỏ nhạt) | Phản bác |
