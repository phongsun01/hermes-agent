# Modern Blue — Bộ màu hiện đại

Dùng cho đề xuất doanh nghiệp nhỏ/vừa, startup, fintech, báo cáo nội bộ.

---

## 1. Bảng màu chính

| Vai trò | Mã HEX | Mô tả |
|---|---|---|
| Primary | `#1565C0` | Blue — heading, header bảng |
| Accent | `#0D47A1` | Blue đậm — nhấn mạnh |
| Secondary | `#616161` | Xám — subtitle, caption |
| Body text | `#000000` | Đen thuần |
| Border / line | `#BDBDBD` | Xám nhạt |

---

## 2. Áp dụng cho Heading

| Cấp | Color | Style |
|---|---|---|
| Heading 1 | `#0D47A1` (Accent) | Bold |
| Heading 2 | `#1565C0` (Primary) | Bold |
| Heading 3+ | `#000000` (Body) | Bold |
| Caption | `#616161` (Secondary) | Italic |

---

## 3. Áp dụng cho Bảng

| Vai trò | Mã HEX |
|---|---|
| Header fill | `#1565C0` (Primary) |
| Header text | `#FFFFFF` |
| Zebra row lẻ | `#F5F5F5` |
| Zebra row chẵn | `#FFFFFF` |
| Dòng tổng | Bold, không fill |

---

## 4. Áp dụng cho Callout / Block đặc biệt

| Loại | Màu chữ | Ghi chú |
|---|---|---|
| Nguyên tắc | `#1565C0` (Primary) | Bold italic |
| Điểm mấu chốt | `#0D47A1` (Accent) | Bold italic |
| Lưu ý | `#616161` (Secondary) | Italic |

---

## 5. Màu phân loại (Traffic Light)

| Cấp | Fill | Ý nghĩa |
|---|---|---|
| Safe / Cấp 1 | `#E8F5E9` (lá nhạt) | An toàn |
| Warn / Cấp 2 | `#FFF8E1` (vàng nhạt) | Thận trọng |
| Danger / Cấp 3 | `#FFEBEE` (đỏ nhạt) | Cấm |
